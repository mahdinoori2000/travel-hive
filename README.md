# 📖 Travel Hive <a name="about-project"></a>


**Travel Hive** is a user-friendly web application designed specifically for travel agencies. Whether you're looking to book hotels, cars, flights, or cruises, our platform has you covered. Simply sign up or log in to access a seamless booking experience. We seamlessly integrate with various APIs to provide up-to-date data for cars, hotels, flights, and cruises, ensuring you have the latest information at your fingertips.

## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to run:

- Terminal
- Code editor like VS code

### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git https://github.com/mahdinoori2000/travel-hive.git
```


### Install

Install this project with:


```sh
  cd travel-hive
  npm install
```


### Usage

To run the project, execute the following command:
